How to use this in Voice Dream Reader

  • Open Voice Dream → Add (+) → Web Content → Manage Sources → + Add New Source.
  • Paste the public URL where you host this page (Google Drive, Notion, GitHub Pages, etc.).
  • Save. Now you have one master link that opens this list inside Voice Dream.
Tip: For feeds that update (like news), you can also add their RSS links directly to Voice Dream for auto-refreshing content.
